# Movieflix API 🎬

Interface de backend de alta performance desenvolvida para demonstrar conceitos avançados de engenharia de software e escalabilidade.

### 🛠 Tecnologias
- **Runtime:** Node.js
- **Linguagem:** TypeScript (Tipagem forte e segurança de dados)
- **Framework:** Express
- **ORM/DB:** Prisma / PostgreSQL (ou MySQL)

### 🚀 Funcionalidades & Destaques
- **Arquitetura Escalável:** Estrutura de pastas e código preparada para crescimento e fácil manutenção.
- **Segurança de Dados:** Validação rigorosa de inputs e tratamento de erros.
- **Documentação:** API documentada seguindo padrões de mercado.
- **Performance:** Otimização de queries para garantir respostas rápidas.
